== Tools ==

This is a reminder of how to install smith from the Ubuntu repositories directly.
On Ubuntu directly or in an Ubuntu VM hosted on another OS.


The repositories (or PPA: Private Package Repositories) to add are:


sudo add-apt-repository ppa:smith-dev/smith
sudo add-apt-repository ppa:fontforge/fontforge

sudo apt-get update
sudo apt-get upgrade

sudo apt-get install smith fontforge


Other utilities are:

sudo apt-get install mercurial git gitg tortoisehg

Optional but very useful packages include:

sudo apt-get install meld tortoisehg-nautilus subversion-tools nautilus-open-terminal


Other SIL software is available from the main SIL repository for Ubuntu:

wget http://packages.sil.org/sil.gpg
sudo apt-key add sil.gpg
sudo add-apt-repository "deb http://packages.sil.org/ubuntu trusty main"
